¿Qué es la Web?

La World Wide Web es un sistema de documentos de hipertexto vinculados a los que accedemos a través de Internet. Con un navegador web, un usuario puede visualizar sitios web que contienen texto, imágenes, videos u otros contenidos multimedia, navegando entre ellos mediante enlaces.
